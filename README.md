# 羽球預約管理系統

會員管理、場地預約、公告管理、臨打活動、商品販售、訂單管理

## 環境設定

### 1. 安裝 Tomcat
- 下載 [Apache Tomcat 10.1](https://tomcat.apache.org/download-10.cgi)
- 解壓縮到專案根目錄，資料夾命名為 `apache-tomcat-10.1.52`（或對應版本）

### 2. Eclipse 設定
1. 開啟 Eclipse，Workspace 選擇 `BadmintonWorkspace`
2. **Window → Preferences → Server → Runtime Environments → Add...**
3. 選擇 **Apache Tomcat v10.1**，指向解壓縮的 Tomcat 資料夾
4. 匯入專案：**File → Import → General → Existing Projects into Workspace**
